# 문제
+ 주어진 문자열에서 반복되지 않는 첫 번째 문자를 반환하는 프로그램을 작성하는 문제입니다.



# 풀이
+ 처음에는 Set을 사용하여 중복되지 않은 문자를 찾으려 했습니다.<br/>
하지만 Set은 중복된 문자를 찾기에 적합하기에 HashMap을 사용했습니다.
+ HashMap에 getOrDefault()를 사용하여 처음 들어오는 데이터는 1의 값으로<br/>
중복되는 데이터는 원본 데이터에 1을 더한 값을 추가했습니다.
+ HashMap을 사용한 이유는 Hash를 사용하여 인덱스를 구분하기 때문에<br/>
임의 접근 O(1)의 시간 복잡도를 가지며 Key 값은 중복되지 않기 때문입니다.<br/>
또한 Value 값은 중복 여부가 상관 없어 getOrDefault()로 임의의 중복 값을 설정 가능합니다.
